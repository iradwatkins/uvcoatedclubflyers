## Quick Start Guide

1. **Starting a new project:**
   - Run `/analyst` then use `*workflow-init` to begin the structured workflow

2. **Working on features:**
   - Use `/pm` to create PRD and break into stories
   - Use `/architect` for technical design
   - Use `/dev` to implement

3. **Get workflow status at any time:**
   - Any agent can run `*workflow-status` to see where you are

4. **Collaboration:**
   - Use `*party-mode` from any agent to bring multiple agents together

## BMad Method Philosophy

- **Scale-Adaptive:** Automatically adjusts process depth based on project size
- **Agent-Driven:** Specialized AI agents with domain expertise
- **Workflow-Guided:** Built-in best practices and structured processes
- **Context-Aware:** Maintains project context across agents
